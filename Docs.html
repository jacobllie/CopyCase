CopyCase.py

Overview

The CopyCase.py script is a Python script that copies a patient case in RayStation. It exports images, beams, doses, and plans from an existing case and imports them into a new case with a specified destination folder. The script requires the plans to be unapproved for it to work correctly.

Dependencies

This script relies on the following dependencies:

  • connect: This module provides access to RayStation's scripting interface.

Usage

To use the CopyCase.py script, follow these steps:

  1. Ensure that the plans you want to copy are unapproved.
  2. Run the script by executing the Python file using the Python interpreter.
  3. Specify the destination folder where the copied case will be created by setting the destination variable in the script.
  4. Set the delete_folder variable to True if you want to delete existing files and folders in the temporary export folder. Set it to False if you want to keep them.
  5. Execute the script and wait for it to complete the copying process.

get_parameters_and_export.py

Overview

The get_parameters_and_export function is responsible for extracting RayStation case parameters and exporting examinations, plans, and doses. It saves the extracted parameters as JSON files and exports the necessary data using the DICOM export function.

Dependencies

This function relies on the following dependencies:

  • json: This module provides JSON serialization and deserialization.
  • os: This module provides a way to interact with the operating system.
  • get_and_set_arguments_from_function: This local file contains functions to extract and set arguments from a function.
  • dicom_export: This local file contains the DICOM export function.

Usage

To use the get_parameters_and_export function, follow these steps:

  1. Call the function and pass the necessary parameters: initials, destination, patient, and case.
  2. The function will extract isodose colortable, imaging scan names, optimization objectives, clinical goals, and plan CT names.
  3. It will save the extracted parameters as JSON files in the specified destination folder.
  4. Finally, it will export examinations, plans, and doses using the DICOM export function.

  • initials (str): The initials to be used in filenames.
  • destination (str): The path of the destination folder where the extracted parameters and exported data will be saved.
  • patient (RayStation PyScriptObject): The RayStation object representing the patient.
  • case (RayStation PyScriptObject): The RayStation object representing the case.

Return Value

This function does not return any value (None).
